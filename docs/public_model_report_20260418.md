# 対外向けモデル状況レポート 2026-04-18

## 1. このレポートの目的

このレポートは、2026-04-18 時点の `nr-learn` のモデル状況を、対外説明に使える形で整理したものである。

本レポートの目的は次の 3 点である。

1. JRA について、何が確認できていて、何がまだ未確定かを分けて示す。
2. NAR について、JRA とは別系統の比較対象として現在地を示す。
3. 指標の名称だけを並べるのではなく、それぞれが何を測る指標で、現在の数値をどう解釈すべきかを明示する。

## 2. このレポートで使う用語

このレポートでは、内部で使っている用語をそのまま捨てるのではなく、意味を定義したうえで使う。

| 用語 | このレポートでの意味 |
| --- | --- |
| JRA | 中央競馬の評価系統。本リポジトリの主要な評価対象である。 |
| NAR | 地方競馬の評価系統。JRA と同じ基準では混ぜず、別系統として扱う。 |
| 運用基準線 | 現時点で実運用の基準として置いている構成。内部識別子では `current_recommended_serving_2025_latest` に相当する。 |
| 正式通過候補 | 検証上の判定を通過した候補。すぐに運用基準線へ置き換えることを意味しない。 |
| 回収率 | 賭け金に対してどれだけ払い戻しがあったかを示す指標。`1.0` が収支とんとん、`1.0` 未満は赤字、`1.0` 超は黒字である。 |
| 期間をずらした繰り返し検証 | 学習期間と検証期間を何度もずらして繰り返す検証。特定期間だけの偶然ではないかを見るために使う。 |
| 買い件数 | 実際に「買う」と判定された件数。少ないほど回収率がぶれやすい。 |

## 3. 先に結論

現時点の結論は次のとおりである。

- JRA は、馬を上位に並べる性能そのものは維持している。
- ただし JRA は、実際の買いを伴う収益性については、まだ強く断言できる段階ではない。
- NAR は、比較の基準となる参照線は再構築できている。
- ただし NAR は、その参照線を安定して上回る新しい主力構成をまだ作れていない。

したがって対外説明としては、「JRA は順位付け性能を維持しているが、収益性はまだ慎重に評価すべき段階である。NAR は比較基準の整備には到達したが、新しい主力構成の確立は継続中である」と述べるのが妥当である。

## 4. 評価の読み方

本レポートでは、指標を次の 2 段に分けて読む。

1. 馬をどれだけ適切に上位へ並べられているか。
2. その順位付けを使って、実際の買いで収益が出ているか。

この 2 つは同じではない。順位付けがうまくても、配当や買い条件の設計次第で収益は悪化しうる。そのため、順位付け指標と回収率指標は分けて説明する必要がある。

また、回収率は少数の高配当に強く引っ張られるため、買い件数が少ない場合は見かけ上の良い数字でも慎重に読む必要がある。

さらに重要なのは、実際の運用判断は「勝ちやすい馬を上に置く」ことだけでは決まらない、という点である。

馬券の買い判断で必要なのは、単純な勝率上位ではなく、次の 2 つの差をどう扱うかである。

1. モデルが見積もる勝ちやすさ。
2. 市場が織り込んでいる価格、つまりオッズ側の見積もり。

したがって、順位付け指標は重要ではあるが、それ自体が最終目的ではない。順位付け指標は「土台の予測が壊れていないか」を見るための下位指標であり、最終的に見たいのは「市場との差分を使って、どの条件で買うと長期的に期待値が残るか」である。

この意味で、`top1 ROI` は主目的そのものではなく、あくまで補助的な診断値である。最上位馬を毎回買う単純ルールは実運用の買い方ではないため、この値だけを上げ続けても長期のモデル最適化に直結するとは限らない。

## 4.1 指標の階層

本レポートでは、指標を次の 3 層に分けて理解する。

| 層 | 何を見るか | 何のために使うか |
| --- | --- | --- |
| 予測の土台 | AUC などの順位付け指標 | モデルが馬の強弱をある程度正しく捉えているかを確認する。 |
| 市場との差分 | 期待値条件付き回収率、市場情報を含む確率の差 | どこに価格のゆがみがありうるかを確認する。 |
| 運用の実装 | 期間をずらした繰り返し検証での回収率、買い件数、買い頻度 | 実際の買いルールが長期に耐えるか、特定局面への過剰適合でないかを確認する。 |

この順序が重要である。土台の予測が弱ければ市場との差分も安定しない。一方で、土台の予測がそこそこ良くても、市場との差分をうまく取れなければ収益にはつながらない。

そのため、長期的な最適化を `top1 ROI` の微修正だけに寄せるべきではない。根本的に改善したいなら、少なくとも次の順で見直す必要がある。

1. どの特徴量で市場が見落としている情報を表現できるか。
2. どの予測モデルでその情報を安定して確率へ変換できるか。
3. その確率と市場オッズの差を、どの買いルールで収益へ変換するか。

したがって、枝葉の閾値調整だけで長期最適化を進めるのは限界がある。閾値調整は最後の運用層で効くが、予測の土台や市場との差分の表現が弱い場合、それだけで持続的な改善を作るのは難しい。

## 5. JRA の状況

### 5.1 位置づけ

JRA の現在の運用基準線は `current_recommended_serving_2025_latest` である。これは正式判定を通過しており、現時点で採用している基準構成である。

一方で、`r20260326_tighter_policy_ratio003` のような正式通過候補も存在する。ただし、正式通過候補は「検証上は有望」という意味であり、直ちに運用基準線へ置き換えることを意味しない。

### 5.2 指標の定義と現在値

| 指標名 | 定義 | 現在値 | 実務上の意味 |
| --- | --- | ---: | --- |
| AUC | 全出走馬を比較したとき、上位に置いた馬ほど実際にも上位に来やすいかを表す指標。`0.5` に近いとほぼ無作為、`1.0` に近いほど順位付けが良い。 | `0.8401` | 順位付け能力そのものは高い。少なくとも「どの馬を上に置くか」という基礎能力は維持されている。 |
| top1 ROI | 各レースでモデルが最上位に置いた 1 頭を機械的に毎回買ったときの回収率。 | `0.8070` | これは主目的ではなく、単純な順位付けをそのまま買いに使うとどうなるかを見る診断値である。現状では、その単純運用は赤字である。 |
| EV top1 ROI | 買い条件を満たしたレースだけで、最上位の 1 頭を買ったときの回収率。内部では期待値条件付きの運用近似として扱っている。 | `0.5568` | 市場との差分を利用する買い条件まで含めても、現行の運用層はまだ十分に利益局面を拾えていない。 |
| nested WF weighted ROI | 期間をずらした繰り返し検証ごとの回収率を、検証量に応じてまとめた加重平均。単発の好不調ではなく、時期をまたいだ再現性を見るための値。 | `0.7628` | 時期をずらしても平均では赤字であり、現行構成を収益構成と呼ぶのは難しい。慎重運用が妥当である。 |
| bets | 回収率計算の中で、実際に「買う」と判定された件数。 | `544` | 回収率を支える件数が少ない。少数の高配当で全体が大きく動く余地があり、過学習を完全には否定しにくい。 |
| test races | 期間をずらした繰り返し検証で評価対象になったレース数。 | `16215` | 順位付け能力を見るための母数は一定程度ある。一方で、この分母に対して買い件数は約 `3.35%` と低い。 |

### 5.3 現在の数値をどう解釈するか

JRA では、順位付け能力と収益性を分けて読む必要がある。

まず、AUC `0.8401` という値からは、馬を上位に並べる能力そのものは維持されていると読める。これは「モデルの土台」が崩れていないことを示す。

しかし、ここから直ちに「買いで勝てる」とは言えない。`top1 ROI = 0.8070` は、勝率上位をそのまま買う発想では赤字になることを示している。さらに `EV top1 ROI = 0.5568` は、市場との差分を使う買い条件の層まで含めても、まだ十分な収益性に届いていないことを示す。

また、`nested WF weighted ROI = 0.7628` は、期間をまたいで見ても平均では赤字であることを示している。単発の良い期間だけを切り出した結果ではなく、時期をずらしてもなお収益性が弱いという点が重要である。

したがって、JRA の現状課題を `top1 ROI` の調整問題としてだけ捉えるのは不十分である。より重要なのは、「市場との差分を作れる特徴量が足りているか」「その差分を確率へ変換するモデルが適切か」「その差分を買いへ変換する運用層が過剰適合していないか」という 3 層を分けて見直すことである。

### 5.4 統計的にどこまで言えるか

JRA については、次のように分けて述べるのが客観的である。

- 順位付け能力については、`n_rows=120000`、`n_races=8702`、さらに期間をずらした検証で `16215` レースを見ているため、ある程度の裏付けがある。
- ただし、収益性については `544` 件しか買いが発生しておらず、回収率が少数の高配当や特定時期の当たりに引っ張られる余地が大きい。
- そのため、「順位付け性能には一定の裏付けがあるが、収益性はまだ慎重に見るべき段階」と表現するのが妥当である。

### 5.5 正式通過候補の位置づけ

`r20260326_tighter_policy_ratio003` は、正式比較での加重回収率 `1.1728` を示しているため、難しい局面に対する守備的な候補として意味がある。

ただし、この候補も買い件数は約 `500` 件台であり、買い頻度は低い。したがって、「正式に通過したこと」は言えても、「長期の主力構成として十分に確認された」とまでは言えない。

### 5.6 JRA で次に本当に見直すべきもの

JRA で長期的な改善を狙うなら、優先順位は次のとおりである。

1. 市場とモデルの差分を作れる特徴量を増やすこと。
2. 勝率予測モデルそのものの置き方を見直し、価格差へつながる確率表現を改善すること。
3. 最後に、その差分を運用へ変換する閾値や買い条件を調整すること。

逆に言えば、最終段の閾値調整だけを続けても、土台の予測や市場との差分表現が弱いままでは改善余地に限界がある。対外説明でも、この点を「予測の土台は確認できているが、収益化の本体はまだ改善余地が大きい」と表現するのが適切である。

## 6. NAR の状況

### 6.1 位置づけ

NAR は JRA と同じ基準では混ぜず、別系統の比較対象として扱う。

現在、信頼できる比較基準として扱っているのは `#101` の正式再実行版である。これは「NAR の比較基準を履歴データ上で読み直せる状態」までは到達したことを意味する。

一方で、次世代候補として進めた `#103` value-blend bootstrap は、この比較基準をまだ上回れていない。

### 6.2 比較基準 `#101` の指標定義と現在値

| 指標名 | 定義 | 現在値 | 実務上の意味 |
| --- | --- | ---: | --- |
| AUC | 上位に置いた馬ほど実際にも上位に来やすいかを表す順位付け指標。 | `0.8625` | 順位付け能力はかなり強い。ただし JRA と市場構造が違うため、単純比較はできない。 |
| top1 ROI | 各レースで最上位の 1 頭を毎回買ったときの回収率。 | `0.8402` | これは単純運用の診断値であり、NAR でも勝率上位をそのまま買えば赤字である。 |
| EV top1 ROI | 買い条件を満たしたレースだけで最上位の 1 頭を買ったときの回収率。 | `2.1111` | 市場との差分を使う運用層では強い数字が出ているが、その強さのかなりの部分は市場情報依存である。 |
| nested WF weighted ROI | 期間をずらした繰り返し検証での加重平均回収率。 | `3.9661` | 数値は非常に強いが、高いからこそ市場依存や履歴条件を含めて慎重に読む必要がある。 |
| bets | 実際に買いが発生した件数。 | `778` | JRA よりは厚いが、なお一部の高配当で全体が押し上がる可能性は残る。 |

### 6.3 現在の数値をどう解釈するか

NAR の比較基準 `#101` は、順位付け指標も条件付き回収率も強い。

ただし、この強さは市場情報を強く取り込む構成の上に成り立っている。市場情報を外すと性能が大きく落ちることが確認されているため、「モデル単体の純粋な予測力」として JRA と同じ文脈で語るべきではない。

加えて、履歴データについては「その時点で本当に利用可能だった情報か」という点にまだ注意が残る。したがって、NAR の高い回収率は、外向けには「強い参照値だが、なお慎重に読むべき値」と位置づけるのが客観的である。

### 6.4 次世代候補 `#103` の現在地

次世代候補 `#103` value-blend bootstrap の主要値は次のとおりである。

| 指標名 | 定義 | 現在値 | 実務上の意味 |
| --- | --- | ---: | --- |
| AUC | 順位付け能力を表す指標。 | `0.7386` | 比較基準 `#101` を大きく下回る。 |
| top1 ROI | 最上位馬を全件買いしたときの回収率。 | `0.7956` | 比較基準より弱い。 |
| nested WF weighted ROI | 期間をずらした繰り返し検証での加重平均回収率。 | `0.6579` | 比較基準を明確に下回っている。 |
| promotion gate decision | 正式昇格判断。 | `hold` | 現時点では昇格不可である。 |
| fast gate feasible folds | 早期判定で成立した期間分割数。 | `0` | 早い段階でも成立性を示せていない。 |

このため、NAR は「比較基準の再構築には成功したが、それを安定して上回る新構成はまだできていない」と整理するのが妥当である。

## 8. 対外説明での推奨表現

対外向けには、次の表現が最も誤解が少ない。

> JRA は、馬を上位に並べる性能そのものは維持している。ただし、実際の買いで安定した利益が出ると断言できるだけの裏付けはまだ薄く、収益性は慎重に評価すべき段階である。NAR は、比較基準となる参照線の再構築には成功しているが、それを安定して上回る新しい主力構成はまだ確立していない。
